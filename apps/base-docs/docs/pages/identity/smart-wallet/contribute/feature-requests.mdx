# Feature Requests

Thank you for your interest in contributing to Smart Wallet.
We welcome feature requests and chain requests!

Please submit your feature requests to one of the following channels:

- [Coinbase Developer Platform Discord](https://discord.gg/smartwallet)
- [Smart Wallet GitHub Issues](https://github.com/coinbase/smart-wallet/issues)

## Feature Request Guidelines

- Please be specific about the feature you'd like to see.
- Please be respectful and constructive.
- Please be patient as we review your request.
